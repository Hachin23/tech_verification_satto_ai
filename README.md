# 技術検証： カメラ検証（iOS Safari & iOS PWA）

## 1. 検証の目的
- 卒業制作アプリ Satto_ai はカメラ撮影がUXの中心なので、実装段階で想定通りに動かないと仕様変更のリスクが生じる為

## 2. 検証項目
  ### Step1 カメラプレビューが出るか
  - getUserMedia()が動くか
  - 縦横切り替えで崩れないか
  - 権限ダイアログが出るか

  ### Step2 ホーム画面追加（PWA）で動くか
  - カメラが真っ黒にならないか
  - 権限がSafariと別扱いにならないか
  - 再起動で動作が変わらないか

  ### Step3 撮影 → Canvasリサイズ → サーバー送信
  - 画像サイズが重すぎないか
  - iOS SafariでUIが固まらないか

  ### Step4 取り直しループのUXが成立するか
  - カメラ → 診断(ダミー診断) → 取り直し
